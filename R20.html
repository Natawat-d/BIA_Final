Main Argument

A Diagnostic Learning Analytics (DLA) model built on distributed clustering plus pose estimation can mine classroom behaviour data to diagnose learning problems — going beyond the static grade data that traditional learning analytics relies on.

Problem / Context

Traditional Learning Analytics relies on static data (assignment submissions, transcripts) and misses real behavioural engagement in the classroom. At the same time, big-data scale (classroom video across many students and sessions) demands distributed processing rather than a single-machine pipeline.

Data & Method

The model combines M-Canopy-K-means clustering — using Canopy pre-clustering to choose good initial centroids, parallelised with MapReduce — with the PSSHRNet pose-estimation network, which recognises classroom behaviour from video. Both components are benchmarked on standard pose-estimation datasets, MSCOCO and CrowdPose, before being applied to real classroom behaviour data.

Key Results

The clustering step achieves +10% accuracy vs plain K-means and +7% vs mini-batch K-means, with a speedup ratio > 2× on small-scale datasets. PSSHRNet achieves lower loss and faster convergence than baseline pose-estimation approaches. Applied to a real classroom, the model finds an engagement participation score of only 2.81, with mobile-phone use accounting for 33.29% of observed classroom behaviour — triggering targeted, personalised intervention.

5 Lessons Learned

  1. Big-data analytics reaches into education: behaviour data can diagnose learning problems that grade data alone hides.
  2. MapReduce parallelisation is what makes clustering feasible at classroom-video scale — a direct application of Big Data lecture concepts.
  3. Better centroid initialisation (Canopy pre-clustering) fixes a classic weakness of K-means, improving both accuracy and speed simultaneously.
  4. Combining unsupervised clustering with deep learning (pose estimation) outperforms either technique used alone.
  5. Analytics must end in action: the diagnosis (33.29% phone use) is designed to trigger a concrete, personalised teaching intervention.
