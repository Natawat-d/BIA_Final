Main Argument

Exposing an analytics platform’s APIs as tools callable by LLM-based AI agents (function calling) lets analysts run whole analytical workflows in natural language — a concrete step toward autonomous data analytics.

Problem / Context

Traditional form- and API-based analytics platforms impose real cognitive overhead and technical barriers. Non-expert users cannot drive them directly, so useful insight ends up bottlenecked on a small pool of technical specialists.

Data & Method

The authors convert the analytics platform’s APIs into LLM function-calling tools. A coordination process mediates between the data analyst, the AI agents, and the platform, producing a chat-based interface for running analytical workflows, instantiated using open-source models. The system is evaluated with a user study that compares the new chat interface against the original form-based interface, measuring task time, number of actions, usability, satisfaction, confidence, and hallucination.

Key Results

Task time falls from 6.71 to 2.70 minutes; the number of actions falls from 14.48 to 3.30; usability rises from 3.09 to 4.64; and the chat interface achieves a +7.2% mean overall score versus other baselines. The chat interface scored better by at least one point on usability, ease of use, satisfaction and timeliness, and notably reduced the “familiarity barrier” — users no longer needed to already know the platform or the specific use case to get results.

5 Lessons Learned

  1. Natural language is emerging as the next BI interface — it collapses the cost of learning a platform’s tools and menus.
  2. Function calling / tool use is precisely what turns a general chatbot into an agent that actually executes analytics workflows.
  3. The measurable win is time and effort (roughly 2.5× faster, about 4× fewer actions), not just subjective user delight.
  4. GenAI democratises analytics — non-experts can now run workflows that used to require specialist platform knowledge.
  5. Risks remain real: hallucination and trust must be explicitly measured, not simply assumed away by a friendlier interface.

Exam focus

Exam angle: “What does ‘autonomous data analytics’ actually mean, and what evidence supports it?” Answer: platform APIs exposed as LLM function-calling tools let a chat interface replace form-based analytics; a controlled user study shows task time drops 6.71→2.70 min and actions drop 14.48→3.30, with usability up and the familiarity barrier reduced — while hallucination risk is explicitly measured, not ignored.
